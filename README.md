# Gratia — Daily Devotional & Bible Study App

A simple and beautiful app for daily bible readings, reflection plans, and prayer journals.

---

## What's inside?

- **Home Dashboard**: Streak tracking, Daily Verse, and reading plans.
- **Bible Reader**: Read scriptures, highlight verses, take notes, and share.
- **Audio Reader**: Listen to scripture read-alouds.
- **Prayer Journal**: Keep track of your prayers and write down answers.
- **Community Feed**: Connect and share encouraging words with friends.
- **Trivia Game**: Bible-themed quizzes to test your knowledge.

---

## How it's organized

This project is split into clean folders:
- **`public/`**: Everything that runs in your browser (HTML page, styles, code files, and images).
- **`server/`**: The backend server code.
- **`data/`**: Where your local user accounts and data are saved.
- **`docs/`**: Extra files and documentation.

---

## How to Run It

To run the app on your computer, make sure you have **Node.js** installed, then follow these steps:

1. Open your terminal (Command Prompt or PowerShell on Windows) in this folder.
2. Run this command to install the project dependencies:
   ```bash
   npm install
   ```
3. Run this command to start the app:
   ```bash
   npm start
   ```
4. Open your web browser and go to:
   ```text
   http://localhost:3000
   ```
